HTML

  • HTML provides content structure or frame
  • Tag is element inside <>
  • The head element contains info about the webpage.
  • The body element represents visible content shown to user
  • Attribute is additional info @ element, defined in beginning, ex. class/id

CSS

  • Selector { property: value; }
  • margin is space outside element
  • padding adds space around content inside element
  • DRY code - combine selectors into single rule

Git

  • git status: checks current working branch
  • git checkout -b branch-name: creates new branch and switches to it
  • Merge: git status, git add -A, git commit -m "message", git pull origin main,git push origin feature/"name feature"
